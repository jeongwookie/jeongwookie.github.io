이번에는 조금 다른 형태의 입력값을 알아보자.

man=True와 같이 매개변수에 미리 값을 넣어준 경우, 따로 입력하지 않아도 자동으로 True값으로 고정된다.

그래서 두 가지 결과값이 동일하게 출력된 것이다.

마지막으로, 함수 사용 시 주의할 점을 소개한다.
